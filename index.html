关于工脉智感

工脉智感面向船舶柴油机高压油管，使用外贴式柔性电容传感器感知喷油脉动引起的管壁微振动与微形变。系统通过 STM32、PCAP01 与 HC-05/串口完成 C1/C2/C3 三通道采集，在 Android 和 Windows 端呈现连续“喷射心电图”，辅助多缸一致性比较、维修前后对比和异常趋势提示。

说明：本系统测量的是油管外壁响应，不直接等同于管内绝对压力或独立故障诊断结论。

安装说明

Android 通用版

  1. 下载 APK 到 Android 手机或平板。
  2. 如系统提示，允许浏览器或文件管理器安装未知应用。
  3. 完成安装后，先在系统蓝牙中与 HC-05 配对。
  4. 打开“工脉智感”，授予蓝牙和通知权限后连接。

Windows 电脑端

  1. 下载 EXE 到 Windows 电脑。
  2. 首次运行如出现安全提示，确认文件校验值后选择继续。
  3. 确认电脑已启用 .NET Framework 4.8。
  4. 选择蓝牙串口或 USB 串口后开始采集。
